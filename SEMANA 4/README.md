Tarea Semana 4 - Desarrollo de Aplicaciones Web.

Descripción
Este sitio web fue creado utilizando HTML5, CSS3 y Bootstrap.
Incluye una estructura básica con navegación, diseño responsivo,
formulario de contacto y estilos personalizados.

Tecnologías utilizadas
- HTML5
- CSS3
- Bootstrap 5

Contenido del sitio
- Encabezado con título principal
- Barra de navegación responsiva
- Sección principal con grid de Bootstrap
- Formulario responsivo
- Botones con animaciones CSS
- Diseño adaptable a dispositivos móviles
